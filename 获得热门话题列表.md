### 获得热门话题列表

> https://api.juejin.cn/tag_api/v1/theme/list_by_hot

**请求方式：POST**

认证方式：无

**URL参数：**

| 参数名 | 类型   | 必传 | 内容 |
| ------ | ------ | ---- | ---- |
| aid    | string | 否   | 未知 |
| uuid   | string | 否   | 未知 |

**Body参数：**

| 参数名  | 类型   | 必传 | 内容         |
| ------- | ------ | ---- | ------------ |
| cursor  | string | 否   | 游标         |
| id_type | number | 否   | 未知 默认为4 |
| limit   | number | 否   | 请求数量     |

**响应格式：application/json; charset=utf-8**



**根对象:**

| 字段     | 类型   | 内容              |
| -------- | ------ | ----------------- |
| err_no   | num    | 响应码            |
| err_msg  | string | 响应消息          |
| data     | array  | 响应内容          |
| cursor   | string | 游标-base64字符串 |
| has_more | bool   | 是否还有更多      |

**data对象：**

| 字段         | 类型   | 内容         |
| ------------ | ------ | ------------ |
| theme        | object | 话题信息     |
| recent_users | array  | 最近用户列表 |

**theme：**

| 字段               | 类型    | 内容         |
| ------------------ | ------- | ------------ |
| theme_id           | string  | 话题id       |
| name               | string  | 话题标题     |
| cover              | string  | 话题封面     |
| brief              | string  | 话题描述     |
| is_lottery         | boolean | 是否抽奖     |
| is_rec             | boolean | 未知         |
| rec_rank           | number  | 未知         |
| topic_ids          | array   | 未知         |
| hot                | number  | 未知         |
| view_cnt           | number  | 阅读数量     |
| user_cnt           | number  | 用户参与数量 |
| status             | number  | 状态         |
| ctime              | number  | 创建时间     |
| mtime              | number  | 修改时间     |
| lottery_begin_time | number  | 抽奖开始时间 |
| lotter_end_timr    | number  | 抽奖结束时间 |

**recent_user：**

| 字段                       | 类型   | 内容           |
| -------------------------- | ------ | -------------- |
| user_id                    | string | 用户id         |
| user_name                  | string | 用户名         |
| company                    | string | 用户公司       |
| job_title                  | string | 工作岗位       |
| avatar_large               | string | 用户头像       |
| level                      | num    | 用户等级       |
| description                | string | 用户描述       |
| followee_count             | num    | 关注数         |
| follower_count             | num    | 被关注数       |
| post_article_count         | num    | 发布文章数量   |
| digg_article_count         | num    | 点赞文章数量   |
| got_digg_count             | num    | 文章被点赞数量 |
| got_view_count             | num    | 文章被浏览数量 |
| post_shortmsg_count        | num    | 文章评论数量   |
| digg_shortmsg_count        | num    | 沸点评论数量   |
| isfollowed                 | bool   | 是否关注该用户 |
| favorable_author           | num    | 受欢迎用户排名 |
| power                      | num    | 未知           |
| study_point                | num    | 未知           |
| university                 | object | 未知           |
| major                      | object | 未知           |
| student_status             | num    | 未知           |
| select_event_count         | num    | 未知           |
| select_online_course_count | num    | 未知           |
| identity                   | num    | 未知           |
| is_select_annual           | bool   | 未知           |
| select_annual_rank         | num    | 未知           |
| annual_list_type           | num    | 未知           |
| extraMap                   | object | 未知           |
| is_logout                  | num    | 未知           |
| annual_info                | object | 未知           |
| account_amount             | number | 未知           |
| user_growth_info           | object | 未知           |

```
curl --location --request POST 'https://api.juejin.cn/tag_api/v1/theme/list_by_hot?aid=2608&uuid=7117937868559779362' \
--header 'Content-Type: application/json' \
--data-raw '{
    "cursor": "3",
    "id_type": 4,
    "limit": 10
}'
```
