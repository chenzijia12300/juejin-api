### 掘金文章热榜

#### 请求信息

> https://api.juejin.cn/content_api/v1/content/article_rank

**请求方式：GET**

**认证方式：无**

**URL参数：**

| 参数名      | 类型   | 必传 | 内容                      |
| ----------- | ------ | ---- | ------------------------- |
| aid         | string | 否   | 未知                      |
| uuid        | string | 否   | 未知                      |
| spider      | string | 否   | 未知                      |
| category_id | string | 是   | 请求热榜分类，1为综合类型 |
| type        | string | 是   | 请求类型固定为hot         |

#### 响应信息

**响应格式：application/json; charset=utf-8**



**根对象:**

| 字段    | 类型   | 内容     |
| ------- | ------ | -------- |
| err_no  | num    | 响应码   |
| err_msg | string | 响应消息 |
| data    | array  | 响应内容 |

**data对象:**

| 字段            | 类型   | 内容             |
| --------------- | ------ | ---------------- |
| content         | object | 文章信息         |
| content_counter | object | 文章信息统计数据 |
| author          | object | 作者信息         |
| author_counter  | object | 作者信息统计数据 |
| user_interact   | object | 用户关注类型     |

**content对象:**

| 字段        | 类型     | 内容       |
| ----------- | -------- | ---------- |
| content_id  | string   | 内容id     |
| item_type   | number   | 文章类型   |
| format      | string   | 未知       |
| author_id   | string   | 作者用户id |
| title       | string   | 文章标题   |
| brief       | string   | 文章描述   |
| status      | number   | 状态       |
| ctime       | number   | 创建时间   |
| mtime       | number   | 修改时间   |
| category_id | string   | 分类id     |
| tag_ids     | string[] | 文章标签id |

**content_counter对象:**

| 字段           | 类型   | 内容     |
| -------------- | ------ | -------- |
| view           | number | 浏览数量 |
| like           | number | 点赞数   |
| collect        | number | 收藏数   |
| hot_rank       | number | 热度     |
| comment_count  | number | 评论数   |
| interact_count | number | 互动数   |

**author对象:**

| 字段        | 类型    | 内容     |
| ----------- | ------- | -------- |
| user_id     | string  | 用户id   |
| name        | string  | 用户名   |
| avatar      | string  | 用户头像 |
| is_followed | boolean | 是否关注 |

**author_counter对象:**

| 字段     | 类型   | 内容     |
| -------- | ------ | -------- |
| level    | number | 用户等级 |
| power    | number | 掘力值   |
| follower | number | 粉丝数   |
| followee | number | 关注数量 |
| publish  | number | 未知     |
| view     | number | 未知     |
| like     | number | 未知     |
| hot_rank | number | 未知     |

**user_interact对象:**

| 字段            | 类型    | 内容     |
| --------------- | ------- | -------- |
| is_user_like    | boolean | 是否点赞 |
| is_user_collect | boolean | 是否收藏 |
| is_follow       | boolean | 是否关注 |

**请求例子：**

```
curl --location 'https://api.juejin.cn/content_api/v1/content/article_rank?category_id=1&type=hot'
```
