### 课程列表接口

#### 请求信息

> https://api.juejin.cn/booklet_api/v1/booklet/listbycategory

**请求方式：POST**

**认证方式：无**

**请求头参数：**

| 参数名       | 类型   | 必传 | 内容             |
| ------------ | ------ | ---- | ---------------- |
| cookie       | string | 否   | 用户凭证         |
| content-type | string | 是   | application/json |

**URL参数：**

| 参数名 | 类型   | 必传 | 内容 |
| ------ | ------ | ---- | ---- |
| aid    | string | 否   | 未知 |
| uuid   | string | 否   | 未知 |
| spider | string | 否   | 未知 |

**请求体参数：**

| 参数名      | 类型   | 必传 | 内容                                                         |
| ----------- | ------ | ---- | ------------------------------------------------------------ |
| category_id | string | 否   | 未知                                                         |
| cursor      | string | 否   | 起始游标                                                     |
| sort        | number | 否   | 排序类型<br />全部0<br />最新1<br />热销7<br />降序9<br />升序8<br />全部（默认）10 |
| is_vip      | number | 否   | 是否只看VIP课程，是为1，不是为0                              |
| limit       | number | 否   | 请求数量                                                     |



#### 响应信息



**响应格式：application/json; charset=utf-8**



**根对象:**

| 字段    | 类型   | 内容     |
| ------- | ------ | -------- |
| err_no  | num    | 响应码   |
