### 首页分类短列表-子分类

> https://api.juejin.cn/recommend_api/v1/article/recommend_all_feed

**请求方式：POST**

认证方式：无

**请求格式：application/json**

**URL参数：**

| 参数名 | 类型   | 必传 | 内容 |
| ------ | ------ | ---- | ---- |
| aid    | 字符串 | 否   | 未知 |
| uuid   | 字符串 | 否   | 未知 |

**请求体参数：**

| 参数名      | 类型   | 必传 | 内容                                                         |
| ----------- | ------ | ---- | ------------------------------------------------------------ |
| id_type     | num    | 否   | 官方广告：14<br />普通文章：2                                |
| client_type | num    | 否   | 未知，默认为2608                                             |
| sort_type   | num    | 否   | 推荐：200 <br />最新：300  <br />热榜3天内：3       <br />热榜七天内：7 <br />热榜30天内：30<br /><br />热榜全部：0 |
| cursor      | string | 否   | 游标                                                         |
| limit       | num    | 否   | 请求数量                                                     |
